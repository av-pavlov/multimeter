Дорогие участники!

Чтобы выставить наиболее объективную оценку, проверка ваших решений производится автоматически на наборе тестов, учитывающем различные варианты исходных данных.

На входных данных для каждого теста запускается ваша программа, после чего система сравнивает выходные данные вашей программы с имеющимися у нее эталонными ответами и выносит решение об успешности прохождения теста. За пройденные тесты начисляется определенное количество баллов. Общее количество баллов за каждую задачу — 100. Если в условии задачи не оговорено специально иное, то баллы за каждый пройденный тест начисляются независимо.

Для корректного прохождения процедуры автоматизированного тестирования ваша программа должна строго соблюдать форматы входных и выходных данных, описанные в условии каждой задачи. В частности, не следует печатать никаких приглашений типа «Введите N:». После вывода ответа нельзя использовать функции, ожидающие нажатия клавиши (readkey / _getch / pause), иначе ваша программа будет снята как превысившая лимит времени на тест.

Для ввода-вывода следует использовать стандартные потоки данных (чтение данных с клавиатуры, вывод на экран монитора). Обратите внимание: чтение из файла/запись в файл НЕ ИСПОЛЬЗУЮТСЯ.

Все входные данные при проверке решений предполагаются корректными и удовлетворяющими всем ограничениям, указанным в условии задачи, то есть дополнительно проверять их корректность не нужно.

Максимальное время прохождения теста и максимальный объем используемой памяти — указаны в условиях задач. Если хотя бы один из этих лимитов превышен, решение считается неверным.
